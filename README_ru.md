## 🌟 Обзор

**Evo-ImageAPI** — это высокопроизводительный шлюз API с обратной разработкой, предназначенный для преобразования возможностей генерации изображений ChatGPT Web (chatgpt.com) в стандартизированный SaaS API, совместимый с OpenAI.

Он оснащен передовым **Singularity Kernel**, обеспечивающим самую стабильную в мире активацию конвейера Image 2 (IMG2) с автоматическим диалектическим обнаружением и обходом как для учетных записей Plus, так и для обычных пользователей.

### ✨ Основные характеристики

*   **🔥 Движок IMG2 Dialectic**: анализ протоколов `resolved_model_slug` (gpt-5-3) и `file-service://` в реальном времени для гарантированного качества Image 2 высокого разрешения.
*   **🛡️ Двухэтапный Sentinel 2.0**: реализует последнюю последовательность протокола `/prepare` + `/finalize` (стандарт 2026 года) с автоматическим откатом к одноэтапному процессу.
*   **🎭 TLS Singularity (utls)**: имитирует отпечатки пальцев Chrome 131 с использованием миксинов понижения версии до HTTP/1.1 для обхода обнаружения Cloudflare 429/403 и JA4H.
*   **🔒 Прокси с подписью HMAC**: интегрированный локальный прокси-сервер изображений (`/v1/p/img`) с проверкой подписи HMAC для решения проблем с 403 Forbidden при прямых ссылках на OpenAI.
*   **🏗️ Промышленная панель управления SaaS**: пользовательский интерфейс управления на Vue 3 + TypeScript профессионального уровня с мониторингом кластера в реальном времени и пакетным управлением учетными данными.
*   **🔋 Управление массивным пулом**: поддержка более 10 000 учетных записей с автоматическим восстановлением токенов (RT -> AT) и аудитом состояния.

## 🚀 Быстрый старт

### 1. Требования
*   Go 1.21+
*   Node.js 20+
*   Рабочий прокси (рекомендуется глобальный/США)

### 2. Инструкции по установке

```bash
# Клонировать репозиторий
git clone https://github.com/TokinoChanshi/GPT-image-api.git
cd GPT-image-api

# Настройка бэкенда
cd backend
go mod tidy
cp .env.example .env # Обновите ваши учетные данные
go run scripts/setup_init.go # Инициализация учетной записи администратора
go run main.go

# Настройка фронтенда
cd ../frontend
npm install
npm run dev
```

## 🕵️ Стандарт аудита IMG2

В отличие от устаревших проектов, Evo-ImageAPI идентифицирует **настоящий IMG2** по трем показателям:

1.  **`resolved_model_slug`**: подтверждено как **`gpt-5-3`** или **`i-5-mini-m`**.
2.  **`asset_pointer`**: подтверждено как **`file-service://`** (протокол источника высокого разрешения).
3.  **Размер Payload**: размер файлов стабильно составляет **3.0 МБ - 3.8 МБ** (оригинальное качество).
